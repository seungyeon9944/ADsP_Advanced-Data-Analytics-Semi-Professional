### 3장 데이터 마트
✏️ **이상치** : 실제 오류 인자인지에 대해서는 제거여부 → **실무자 통해서** 결정

✏️ **완전분석법 (Complete Analysis)** : 사례 많을 때는 유용, 결측치 많을 때는 X

✏️ **다중 대치법** : 여러 번 대치하여 여러 개의 가상자료 존재

✏️ **연속형 변수** : 특정조건에 따라 이진화 가능

✏️ **구간화**의 개수가 감소하면 정확도 낮아지고 속도가 높아짐

### 4장 통계 분석
✏️ 모집단 분산 모를 경우 : **자유도가 n-1인 t분포** 이용하여 신뢰구간 측정

✏️ **p-value가 0.05보다 작으면** 0.05 유의수준에서 **존재**, **설명하는데 유의**함 !

✏️ 항상 **귀무가설** = '차이가 없다', '동일하다', '상관관계 없다'

✏️ **비모수검정** : 특정 분포 따른다고 가정 불가 시, **분포의 형태**에 따른 가설 설정

✏️ **비모수 통계** 예시 : 월콕슨, 스피어맨, 맨-휘트니 검정

✏️ **공분산** : 두 변수의 크기와 단위에 따라 값이 달라짐

✏️ **다중공선성** : 독립변수들 간 강한 상관관계

✏️ **결정계수** : 총 제곱의 합 중 **설명된 제곱의 합**의 비율로, 표본 상관계수 r의 제곱값과 같음

✏️ **후진제거법** : 가장 **적은 영향** 주는 변수부터 하나씩 제거하면서 더이상 제거할 변수 없을떄 모형 선택

✏️ **지수평활법** : **최근** 시계열에 **더 많은 가중치** 부여하여 미래 예측

✏️ ARIMA 모형
- **차분** : 평균 일정하지 않을 때
- **변환** : 분산 일정하지 않을 떄

✏️ 다차원척도법 : 동일한 **상대적 거리** 가진 실수 공간의 점들로 배치

✏️ **주성분분석** : 원래 있는 변수와 상관 O, 상관행렬 사용

### 5장 정형 데이터 마이닝

✏️ **텍스트 마이닝** : 문서 분류, 감성 분석, 주제 모델링 등
- 단어사전 : 분석 대상 되는 모든 단어 포함 목록
- 코퍼스(corpus) : 구조화된 단계
- 평가지표 : F1-Score, AUC, ROC 등

✏️ 로지스틱 회귀모형 : 유사성 찾는것과는 별개 !

✏️ **서포트 벡터 머신 (Support Vector Machine, SVM)**
- 생성모델에 대한 해석 어려움
- 학습시간이 오래 걸릴 수 있음
- 기계학습과 관련 but 딥러닝과는 X

✏️ **의사결정나무** : 잡음에 안 민감한 게 장점 !

✏️ 지니지수 : 노드의 **불순도**. 지니지수 = 0일 떄 제일 순수함

✏️ 부스팅 : **잘못 분류**된 데이터에 **더 큰 가중치** 부여

✏️ **활성화 함수** : 출력변수의 속성에 따라 달라짐. 

✏️ **시그모이드** : 신경망 단층으로 구성했을 때 로지스틱 회귀와 유사한 작동 원리

✏️ **재현율** = 민감도

✏️ **최단연결법**
- 거리 = 각 군집에서 하나의 관측값을 뽑았을 때 나타날 수 있는 거리의 최솟값
- 사슬 모양의 군집 생길 수도 있음
- 평균연결법보다 계산량 많음

✏️ 연관분석 : 세분화된 품목, 분석 목표에 따라 규칙 발견 달라짐

✏️ Apriori 알고리즘
- **높은 신뢰도, 최소 지지도** 함께 고려
- **빈발항목집합** (최소지지도라는 임계값 설정, 더 큰 항목 집합 생성)
