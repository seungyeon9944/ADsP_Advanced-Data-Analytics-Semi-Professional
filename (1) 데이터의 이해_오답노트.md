## 1과목 [데이터 이해] 오답노트

### 1장 데이터의 이해
✏️ 데이터 수 != 창출되는 가치

✏️ 텍스트 데이터는 수치 데이터보다 더 많은 자원 소요

✏️ **메타데이터** : 데이터 내부에 있는 건 X. 별도로 저장함 !

✏️ DIKW 피라미드에서 지식 vs 지혜
- **지식**의 예시 : '상대적으로 저렴한 A카페에서 커피를 사야겠다
- **지혜**의 예시 : 'A카페의 라떼도 B카페보다 쌀 것'이라고 판단한다

✏️ 암묵지와 형식지의 상호작용 관계
- 1단계 : **공통화** (대화 통해서 암묵지를 타인에게 알려줌)
- 2단계 : **표출화** (암묵지를 책 등 형식지 - 메뉴얼 보고서 - 로 만들기)
- 3단계 : **연결화** (문서를 결합해 새로운 지식체계 생성)
- 4단계 : **내면화** (학습 통해 적용할 수 있는 방식으로 이해)

✏️ **객체지향 DBMS** : 복잡한 데이터베이스 표현, 관리

✏️ **CRM** (Customer Relationship Management) : 고객 중심 자원 극대화, 탈피와 분석 중심의 시스템 구축 지향

---

### 2장 데이터의 가치와 미래

✏️ 빅데이터의 출현 배경 : **클라우딩 컴퓨팅으로 인한 경제성 확보**

✏️ 빅데이터의 가치 산정이 어려운 이유 : 전문인력의 증가와는 관련 X

✏️ 핀테크 분야에서는 리스크평가를 사유로 **신용평가**에 빅데이터를 사용

✏️ 에너지 산업에서는 **트레이딩, 수요예측, 공급**에 빅데이터 사용

✏️ 데이터 오용 해결방안 : **알고리즘 접근권** 제공

✏️ 비식별 기술의 종류와 예
- **데이터 마스킹** : 특정 데이터값 가려서 처리 (익명)
- **가명처리** : 다른값으로 대체 (규칙 노출 X)
- **범주화** : 대푯값으로 변환하여 숨김

### 3장 가치 창조를 위한 데이터 사이언스와 전략 인사이트
✏️ 가치기반 분석 위한 우선 고려사항

1. 기술 혁신이 사회에 가치창출 영향

2. 주요경쟁사 전략, 시장 점유율 동향

3. 글로벌 경제 및 규제 변화

✏️ 넓은 시야에서 모델 바깥의 요인 판단하는 것은 오히려 과적합을 불러올 수 있음
